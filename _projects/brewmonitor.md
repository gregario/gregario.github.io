Commercial brewing monitors were dear, so I built my own. An Arduino clone, an ESP8266 WiFi module, and a DS18B20 temperature sensor — total cost about 17 euros. It talks to a cloud API so you can monitor your fermentation from anywhere.

Born out of The Bedroom Laboratory, it's got 16 stars on GitHub, which for a niche homebrew hardware project isn't too shabby. The whole point was proving you don't need to spend 200 quid on a fancy controller when a few cheap components and a weekend will do.

[Source on GitHub](https://github.com/thebedroomlaboratory/BrewMonitor)
